# ARIES MLA100 PCIe Card

## 제품 요약

![ARIES MLA100 PCIe Card](/res/image/MLA100-PCIe.jpg)

**ARIES MLA100**은 모빌린트 ARIES NPU를 기반의 PCIe AI 가속 카드입니다. 엣지(Edge) AI 워크로드에 최적화되어 있으며, **25W의 전력 소비**로 최대 **80 TOPS의 추론 성능**을 제공합니다.

## 하드웨어 (호스트 시스템) 요구사항

MLA100 PCIe Card는 AI 가속기로서, 외부 호스트 시스템이 필요합니다.

카드를 설치하고 사용하기 위해서는 다음 조건을 충족해야 합니다.

- 최소 한개의 8-레인 PCIe 슬롯(PCIe X8)을 갖춘 PC 또는 메인보드

## 소프트웨어 요구사항

**운영 체제**

- x86-64 혹은 aarch64 아키텍처
- 우분투 22.04.5 (공식 권장 환경)
- 우분투 20.04, 22.04, 24.04
- 윈도우 10/11 (빌드 19041 이상)
- x86-64 혹은 aarch64 아키텍처 기반의 기타 Linux 배포판도 지원될 수 있습니다.

파이썬 인터페이스를 사용하려면, 시스템에 파이썬이 설치되어 있어야 합니다.

```{note}
공식적으로 지원되는 구성 이외의 환경을 사용하는 경우, tech-support@mobilint.com 으로 문의하세요.
```
